### 斐波那契数

#### 题目

**斐波那契数** （通常用 `F(n)` 表示）形成的序列称为 **斐波那契数列** 。该数列由 `0` 和 `1` 开始，后面的每一项数字都是前面两项数字的和。也就是：

```
F(0) = 0，F(1) = 1
F(n) = F(n - 1) + F(n - 2)，其中 n > 1
```

给定 `n` ，请计算 `F(n)` 。



#### 解题 

1. 确定`dp[i]`含义

   表示第 i 个斐波那契数的值为 `dp[i]`



2. 递推公式

   ```
   dp[i] = dp[i - 1] + dp[i - 2]
   ```



3. dp数组如何初始化



4. 确定遍历顺序

   从前向后

   

5. 打印dp数组
